# iLift - Supabase Features List

## Phase 1: Core (Connect & Basic Sync)
- [ ] Connect Supabase - switch from localStorage to database
- [ ] User accounts - sign up/login with email
- [ ] Persistent profile - stats sync across devices
- [ ] Persistent workout history - past workouts always available

## Phase 2: Social & Competition
- [ ] Live leaderboards - instant rank updates when anyone logs
- [ ] Squad chat - quick messages on squad tab (from our plan)
- [ ] Friend activity feed - see when friends work out
- [ ] Push notifications - "Jordan just passed you!"

## Phase 3: Video & Proof
- [ ] Video upload storage - verify workouts with video
- [ ] Video player in-app - watch proof
- [ ] Badge verification - video proof for badges

## Phase 4: Advanced Features
- [ ] Seasons/competitions - monthly contests
- [ ] Analytics dashboard - trends over time
- [ ] Personal records history - track PRs long-term
- [ ] Export data - download workout history

## Things We Can Build NOW (Local First)
- [ ] Better onboarding flow
- [ ] More workout types/exercises
- [ ] Rest timer improvements
- [ ] UI polish (animations, transitions)

---

## Priority Order Recommendation

1. **First:** Connect Supabase + User accounts (core)
2. **Second:** Live leaderboards (the "competitive" hook)
3. **Third:** Squad chat (social retention)
4. **Fourth:** Video uploads (differentiation)

---

## What's NOT Needed Yet

- Payment/monetization
- Advanced analytics
- API access for third parties
- Complex admin dashboard
